Variance&Conditional Variance

LECTURE 6-7 Discrete Random Variables

Variance

Variance of a random variable is defined as follows:

Var⁡(X)=E[(X−E[X])2]\operatorname{Var}(X)=\mathbb{E}\left[(X-\mathbb{E}[X])^{2}\right] Var(X)=E[(X−E[X])2]

REMARK: E(X)E(X)E(X) is one way to summarize the PMF PXP_XPX​, but it connot capture randomness/uncertainty thus we need variance.

Alternative expresion for variance

Var⁡(X)=E[(X−E[X])2]=E[X2−2XE[X]+E[X]2]=E[X2]−E[2X⋅E[X]]+E[(E[X])2]=E[X2]−2E[X]E[X]+(E[X])2=E[X2]−(E[X])2=E[X2]−E2[X]\begin{aligned} \operatorname{Var}(X) &=\mathbb{E}\left[(X-\mathbb{E}[X])^{2}\right] \\ &=\mathbb{E}\left[X^{2}-2 X \mathbb{E}[X]+\mathbb{E}[X]^{2}\right] \\ &=\mathbb{E}\left[X^{2}\right]-\mathbb{E}[2 X \cdot \mathbb{E}[X]]+\mathbb{E}\left[(\mathbb{E}[X])^{2}\right] \\ &=\mathbb{E}\left[X^{2}\right]-2 \mathbb{E}[X] \mathbb{E}[X]+(\mathbb{E}[X])^{2} \\ &=\mathbb{E}\left[X^{2}\right]-(\mathbb{E}[X])^{2} \\ &=\mathbb{E}\left[X^{2}\right]-\mathbb{E}^{2}[X] \end{aligned} Var(X)​=E[(X−E[X])2]=E[X2−2XE[X]+E[X]2]=E[X2]−E[2X⋅E[X]]+E[(E[X])2]=E[X2]−2E[X]E[X]+(E[X])2=E[X2]−(E[X])2=E[X2]−E2[X]​

Note that Var(X)≥0Var(X)\geq 0Var(X)≥0 and it equals 0 if and only if XXX is a constant function

Conditional variance

conditional variance is analogous to conditional expectation. Let AAA be some event and X,YX,YX,Y be some random variables.

  1. Var⁡(X∣A)=E[(X−E[X∣A])2∣A]=∑x(x−E[X∣A])2⋅PX∣A(x)\operatorname{Var}(X \mid A)=\mathbb{E}\left[(X-\mathbb{E}[X \mid A])^{2} \mid A\right]=\sum_{x}(x-\mathbb{E}[X \mid A])^{2} \cdot P_{X \mid A}(x)Var(X∣A)=E[(X−E[X∣A])2∣A]=∑x​(x−E[X∣A])2⋅PX∣A​(x)

    X∣AX|AX∣A is random variable of XXX

    Alternatively, Var⁡(X∣A)=E[X2∣A]−E2[X∣A]\operatorname{Var}(X \mid A)=\mathbb{E}\left[X^{2} \mid A\right]-\mathbb{E}^{2}[X \mid A]Var(X∣A)=E[X2∣A]−E2[X∣A]

    Similarly, Var⁡(X∣Y=y)=Var⁡(X∣{Y=y})=E[(X−E[X∣Y=y])2∣Y=y]\operatorname{Var}(X \mid Y=y)=\operatorname{Var}(X \mid\{Y=y\})=\mathbb{E}\left[(X-\mathbb{E}[X \mid Y=y])^{2} \mid Y=y\right]Var(X∣Y=y)=Var(X∣{Y=y})=E[(X−E[X∣Y=y])2∣Y=y]

  2. Var⁡(X∣Y)(y)=Var⁡(X∣Y=y)\operatorname{Var}(X \mid Y)(y)=\operatorname{Var}(X \mid Y=y)Var(X∣Y)(y)=Var(X∣Y=y)

    X∣YX|YX∣Y is a random variable of YYY thus a function of YYY

    Var⁡(X∣Y)=E[X2∣Y]−E2[X∣Y]\operatorname{Var}(X \mid Y)=\mathbb{E}\left[X^{2} \mid Y\right]-\mathbb{E}^{2}[X \mid Y]Var(X∣Y)=E[X2∣Y]−E2[X∣Y]

Law of total variance ***

Var⁡(X)=E[Var⁡(X∣Y)]+Var⁡(E[X∣Y])\operatorname{Var}(X)=\mathbb{E}[\operatorname{Var}(X \mid Y)]+\operatorname{Var}(\mathbb{E}[X \mid Y]) Var(X)=E[Var(X∣Y)]+Var(E[X∣Y])

  • why we need the second item?

    consider Var(X∣X)Var(X|X)Var(X∣X) , it is a function of XXX , so Var(X∣X)(x)=Var(X∣X=x)=Var(X=x)=0Var(X|X)_(x)=Var(X|X=x)=Var(X=x)=0Var(X∣X)(​x)=Var(X∣X=x)=Var(X=x)=0, for any xxx the function give the zero output, so we say Var(X∣X)=0Var(X|X)=0Var(X∣X)=0 is a function of constant, the E[Var(X)]=0E[Var(X)]=0E[Var(X)]=0] obviously, if we don’t have the second item the equation above will not make sense.

    PROVE

    E[Var⁡(X∣Y)]=E[E[X2∣Y]−E2[X∣Y]]=E[E[X2∣Y]]−E[E2[X∣Y]]=E[X2]−(Var⁡(Z)+E2[Z])=E[X2]−(E[E[X∣Y]])2−Var⁡(Z)=E[X2]−(E[X])2−Var⁡(Z)\begin{aligned} \mathbb{E}[\operatorname{Var}(X \mid Y)] &=\mathbb{E}\left[\mathbb{E}\left[X^{2} \mid Y\right]-\mathbb{E}^{2}[X \mid Y]\right] \\ &=\mathbb{E}\left[\mathbb{E}\left[X^{2} \mid Y\right]\right]-\mathbb{E}\left[\mathbb{E}^{2}[X \mid Y]\right] \\ &=\mathbb{E}\left[X^{2}\right]-\left(\operatorname{Var}(Z)+\mathbb{E}^{2}[Z]\right) \\ &=\mathbb{E}\left[X^{2}\right]-(\mathbb{E}[\mathbb{E}[X \mid Y]])^{2}-\operatorname{Var}(Z) \\ &=\mathbb{E}\left[X^{2}\right]-(\mathbb{E}[X])^{2}-\operatorname{Var}(Z) \end{aligned} E[Var(X∣Y)]​=E[E[X2∣Y]−E2[X∣Y]]=E[E[X2∣Y]]−E[E2[X∣Y]]=E[X2]−(Var(Z)+E2[Z])=E[X2]−(E[E[X∣Y]])2−Var(Z)=E[X2]−(E[X])2−Var(Z)​

    Thus, E[Var⁡(X∣Y)]+Var⁡(E[X∣Y])=Var⁡(X)\mathbb{E}[\operatorname{Var}(X \mid Y)]+\operatorname{Var}(\mathbb{E}[X \mid Y])=\operatorname{Var}(X)E[Var(X∣Y)]+Var(E[X∣Y])=Var(X)

  • The first step we just expand the variance and the second we use the Linearity of expectation.
  • Then to be concise, let ZZZ denote X∣YX|YX∣Y, the first item in third step become E[X2]E[X^2]E[X2] because the property of smoothing.
  • using smoothing again we get fourth step
  • then let Var(X)Var(X)Var(X) substitute the first two items in step 5, we then get all we want.
