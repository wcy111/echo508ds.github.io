Eternal Sunshine of the Spotless Mind.

阅读分享(贰)

Created2023-01-24Updated2023-01-24

通宵读完的这本小说，读的时候很多情节没有去细想，比如茅大先生和谢晓峰相遇的那段，以及慕容秋荻与谢晓峰的往事。但是总的来说剧情的设置上也没有过于的新奇，许多的坑也没有在结尾填完。比起古龙的其他小说让我觉得此篇只能算的着中规中矩。但是阅读各路书评，回过头思索一些剧情，却发现许多地方值得玩味。

那么首先还是和往常一样，分享阅读过程中比较有意思有感触的片段吧。

一个人思想最乱的时候，通常都很不容易看见别的人，别的事。//

愈平凡的人和事，有时反而愈不容易去不看。

---- 8章 《醉意如泥》

他的声音还是很平静：“每个人都有他自己的命运，是幸运，还是不幸，都怨不上别人，所以这些年来，我也渐渐看开了！”

---- 9章 《深藏不露》

他并不想要别人将他看成英雄，也不想让别人看见他的痛苦。

---- 10章 《剑在人在》

他从不相信任何女人。在他眼中，女人只不过是一种装饰，一种工具，当他需要她们时，她们就会像猫一样乖乖投入他怀里。当他厌倦时，他就会像垃圾般地将她们抛开。

对这一点，他从不隐瞒，也从无歉疚，因为他总认为他天生就应该享受女人的宠爱。

如果有女人爱他，爱得要死，爱得恨不得能死在他怀里，他都认为那女人活该。

所以如果金兰花现在出卖了他，他也会认为自己活该。他一点都不在乎，因为他已经准备拼了。//

金兰花道：“他就是要你们命的阎王！”

这句话没说完，她的人已扑上桌子，用两只手握住桌上的刀，刺入自己的胸膛。

（这段和苏樱换杯饮毒酒的桥段好像）

---- 15章《人事无常》

她的丈夫虽然粗鲁丑陋，他的妻子瘦小干枯，但是他们却能尽量使对方欢愉。因为他们都知道只有这才是他们真正拥有的。他们能有什么，就尽量享受什么。她们对自己的生活很满意。//

生命本就是一片空白，本就要许许多多有价值的事去充实它，其中若是缺少了友情，剩下的还有多少？

---- 18章 《判若两人》

有风，微风。

她抬起手，轻浮被微风吹乱的头发，忽然笑了笑。她很少笑。

她的笑容也像是她的人，美丽、高雅、飘忽，就像春夜中的微风，没有人能捉得住。

她的声音也像是春风般温柔：“已经有很多年了？是十五年？还是十六年？”

他没有回答，因为他知道她一定比她记得更清楚，也许连每一天发生的都能记住。

她笑得更温柔：“看样子你还是没有变，还是不喜欢说话。”

他冷冷地看着她，过了很久，才冷冷地问：“我们还有什么话好说？”

她的笑容消失，垂下了头：“没有了…没有了…"

是不是真的没有了？什么都没有了？//

就算他天生是浪子，就算他血管里流着的都是浪子的血，可是他也有厌倦的时候。

尤其是每当大醉初醒，夜深人静时，又有谁不想身畔能有个知心的人，能叙说自己的痛苦和寂寞？

---- 26章《久别重逢》

这二十年来，他久已学会忍耐。他情愿多费些事，将这些结一个个解开。//

“信心”本身就是种武器，而且是最犀利，最有效的一种。

---- 27章 《聚短离长》

谢掌柜盯着他，反问道：“你真的败了？”

谢晓峰道：“败就是败，真假都一样。”

谢掌柜道：“姑姑就是姑姑，不管她嫁给了什么人都一样。”

谢晓峰道：“你明白就好！”

谢掌柜叹了口气，苦笑道：“明白了也不好，做人还是糊涂些好！”

---- 28章 《身经百战》

不同的城市，同样的人，同样在为着名利和生活奔波。同样要背恩怨情仇所苦。

---- 29章 《患难与共》

阴森的庙宇，沉默的神邸，无论听见多悲惨的事，都不会开口的。

可是冥冥之中却自然又双眼睛，在冷冷的观察者人世间的悲伤和罪恶，真诚的虚假，他自己虽然不开口，也不出手，却自然会借一个人的手，来执行他的力量和法律。

---- 32章 《胸有成竹》

要一个人承认自己的失败，并不是件容易事，到了不能不承认的时候，那种感觉已不仅是羞辱，而且悲伤，一种充满了痛苦和绝望的悲伤。//

一个人到了应该放手的时候还不肯放手，就是自讨无趣了。

---- 34章 《铁骑快剑》

铁开诚道：“陌生人并不可怕。”

因为陌生人既不了解你的感情，也不知道你的弱点。

只有你最亲密的朋友，才知道这些，等他们出卖你时，才能一击致命。//

不管是身上的伤，还是心里的伤，都一样治不了。

谢晓峰却还在笑：“幸好有些伤是根本就不必去治的。”

小弟道：“什么伤？”

谢晓峰道：“根本就治不好的伤。”

小弟看着他，过了很久，才缓缓道：“你醉了。”

谢晓峰道：“你也醉了。”

（这段莫名让我想起唐顿庄园里那句 We all have scars, insde or out）

---- 36章 《欣逢知己》

应该爱的不能去爱，应该恨的不能去恨，对一个血还没有冷的年轻人来说，这种痛苦如何能忍受？

---- 37章 《看破生死》

厉真真垂下头：“一个总是让别人伤心的人，自己也一定会有伤心的时候。”//

谢晓峰道：“你的确不是君子，你是个小人。”

他居然在微笑：“可是真小人至少比伪君子好，真小人还肯说老实话。”

---- 39章《赌剑决胜》

黄昏。有雾。

黄昏本不该有雾，却偏偏有雾。梦一样的雾。

人们本不该有梦，却偏偏有梦。

谢晓峰走入雾中，走入梦中。

是雾一样的梦？还是梦一样的雾？

如果说人生本就如雾如梦？这句话是太俗，还是太真？

---- 41章《看轻生死》

谢晓峰也明白他的意思。有种人好像天生就应该孤独的，这就是他们的命运。

老人慢慢地接着道：“没有人能够改变自己的命运，如果你一定想改变他，结果只有更不幸。”

---- 43章 《了如指掌》

老人并没有吃惊。他已经看见了系在剑锷上的线，也已看见这根线的另一端就在谢掌柜手里。世上有很多不可思议，无法解释的事发生，就因为每件事都有这么样一根线，而人们却看不见而已。

（命运的手，因果的线，皂滑弄人:) ）

谢掌柜道：“因为败过一次后，他才知道自己并不是神，并不是决定不能败的，他一定要受到过这么样一次教训后，才能算真正长成。”

---- 45章 《对手相逢》

人生中本就充满了矛盾，得失之间，更难分得清。

---- 46章 《大惑不解》

他笑得并不勉强：“可是我现在想通了，一个人只要求得心里的平静，无论牺牲什么，都是值得的。”铁开诚沉默了很久，仿佛还在咀嚼他这几句话里的滋味。

然而他又忍不住问：“难道牺牲自己的性命也是值得的？”

谢晓峰道：“我不知道。”

他的声音平和安详：“我只知道一个人心里若不平静，活着远比死更痛苦得多。”

他当然有资格这样说，因为他确实有过一段痛苦的经验，也不知接受过多少次惨痛的经验后，才挣开了心灵的枷锁，得到解脱。

---- 47章 《淡泊名利》

《三少爷的剑》是一部悲剧，书中每个人都有自己的无奈和痛苦。慕容秋荻爱上谢晓峰，因爱生恨，却永远也得不到他。燕十三救人却意外顿悟第十五剑，刻意寻找不可得，真切放下却得到了，是禅意也是讽刺。谢晓峰看破一切，切指归隐却也无法不面对迷失在虚名的挑战者，而他的儿子却已然走在他曾经的道路上…

而通过悲剧古龙想表达的，我认同此书评中的：

英雄都是凡人，凡人就有凡人的苦恼，谁也不能例外，甚至大人物的烦恼比普通人还要大，而且更无解。这就是古龙的意思，也是《三少爷的剑》想要表达的真正内涵。

“人在江湖，身不由己。”人不是神，人要的东西神无法给予，人只有超越自己，战胜自己，才能真正掌握自己命运和心灵，世界总是有太多无奈，我们的人生却都是这样平凡，北岛说生活是网，谁也逃不开，谁也走不掉。

但人却可以超越自己的命运，人可以用坚韧和意志证明人有可以突破自己的命运的决心，燕十三就和《肖申克的救赎》内二十七年如一日救赎自己的安迪一样，他们是自己的神，文学与艺术最应赞誉的成分就应当是这些，因为这股力量可以带给人生活下去的勇气，它伟大，且无处不在。
